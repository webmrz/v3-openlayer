## 开发指南

### 环境要求
- Node.js >= 16
- Vue 3.x
- OpenLayers 7.x
- TypeScript 4.x

### 安装依赖
```bash
npm install
```

### 开发运行
```bash
npm run dev
```

### 构建部署
```bash
npm run build
```
